# Legacy migrations

SQL in this folder is from the **removed Telegram Mini App prototype** (`ldr-couples-app`).

- Tables: `app_users`, `couples`, `touches` (Telegram identity, 1:1 couples only)
- **Do not apply** to a new Ito Supabase project without explicit review
- The target Ito schema (threads, pulses, moments, reactions) is not implemented yet

The prototype is preserved on git branch `backup/pre-ito-telegram-prototype`.
